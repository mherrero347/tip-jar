# tip-jar
a tip jar for the modern era - send me a tip on Venmo, and the tip jar lights up!

# tech we're using in this project
* Python 3.8
* the Unofficial Venmo API by @mmohades
* a Raspberry Pi Zero W
* Adafruit LED light strips
